Land Registry Alpha

The Land Registry are working with The Government Digital Service (GDS) to build a new digital service. The service will form a part of GOV.UK and is currently in Alpha phase following guidance in the Government Service Design Manual in order to meet the Government Digital By Default Service Standard.

The service is being built in the open, with the code developed as open source, and the architecture and other documentation published under the open government licence. Access to the preview systems used for demonstration and testing is currently restricted.

Casework

Entry forms and pages needed to demonstrate the digital service.

GOV.UK

A page for each property registered with the Land Registry, along with search and other tools to appear on GOV.UK.

This page for each property contains public information about the property from across government, and is a starting point for a number of services, including viewing and transacting upon the property using the Land Registry digital service.
